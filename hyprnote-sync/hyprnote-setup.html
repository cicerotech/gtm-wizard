hyprnote setup

AI meeting notes for Eudia

Sales Team
Internal Use
1 Install Hyprnote

Download and install the app.

Download for Mac

Click "Hyprnote" (not Nightly). Drag to Applications. Open and sign in.

Done

Before calls: Hyprnote > New Note > Record

2 Open Terminal

Press Cmd + Space, type Terminal, press Enter.

3 Install Node.js

Paste this command and press Enter:

curl -o ~/Downloads/node.pkg https://nodejs.org/dist/v20.11.0/node-v20.11.0.pkg && open ~/Downloads/node.pkg

Click Continue through the installer. Enter your Mac password if asked.

4 Download Sync Tool

Paste this command and press Enter:

cd ~/Documents && curl -L -o hyprnote-sync.zip https://github.com/cicerotech/gtm-wizard/releases/latest/download/hyprnote-sync.zip && unzip -o hyprnote-sync.zip && rm hyprnote-sync.zip
5 Run Setup

Paste this command and press Enter:

cd ~/Documents/hyprnote-sync && npm install --silent && node setup-quick.js

Enter your Eudia email when prompted. Wait for "Setup Complete".

Done

Meetings sync to Salesforce every 3 hours automatically.

Daily Use
Before call Hyprnote > New Note > Record
After call Stop recording. AI generates notes.
