Step 1: Install App 2 min
1
Download Hyprnote
Download for Mac
  • Click Hyprnote (not "Nightly")
  • Open the .dmg file
  • Drag to Applications
  • Open and sign in
Setup Complete

Open Hyprnote before meetings. Click New Note, then Record.

Step 2: Install Node.js 2 min
2
Required for Salesforce sync
Download Node.js
  • Open the .pkg file
  • Click Continue through all steps
  • Click Install (enter password if asked)
Step 3: Get Sync Tool 1 min
3
Download the sync folder
Download hyprnote-sync.zip
  • Unzip the download
  • Move hyprnote-sync folder to Documents
Step 4: Run Setup 2 min
4
One-time configuration
  • Open Documents > hyprnote-sync folder
  • Double-click install.command
  • Enter your Eudia email when prompted
  • Wait for "Setup Complete"
Blocked by macOS? Right-click the file, click Open, then click Open again.
Done

Meetings sync to Salesforce every 3 hours automatically.

Daily Use
Before call Open Hyprnote > New Note > Record
During call Transcribes automatically
After call Stop recording, AI generates summary
Salesforce Syncs every 3 hours, nothing to do
