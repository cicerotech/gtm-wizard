Quick Setup

1

Install Obsidian

Download from obsidian.md and create a new vault named "Eudia" or similar. This is where your meeting notes will live.

Download Obsidian ‚Üí
2

Install Scribe Plugin

Open Settings ‚Üí Community Plugins ‚Üí Browse ‚Üí Search "Scribe" ‚Üí Install & Enable. This handles voice-to-text transcription and AI summaries.

3

Configure Scribe with OpenAI

In Scribe settings, paste the shared OpenAI API key (provided by RevOps). Select "GPT-4o" as the model. This powers the AI meeting summaries.

4

Create Account Folders

Create a folder for each account you're working. Inside each, add meeting notes as you go. Example: Amazon/2026-01-28 Discovery.md

5

Calendar Sync

Your calendar is already synced via Microsoft Graph. Meeting prep sheets are auto-generated daily for external meetings. Check the Meeting Prep tab.

Recording meetings: In Obsidian, use Scribe's record button during calls. When you stop, it transcribes and generates a summary with action items and next steps. Notes sync to Customer Brain in Salesforce.
