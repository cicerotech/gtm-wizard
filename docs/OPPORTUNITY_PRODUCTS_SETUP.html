Sandbox Setup Steps

  1. 1

    Open Developer Console in Sandbox

    Login to your Sandbox org → Gear icon → Developer Console

  2. 2

    Open Execute Anonymous Window

    Debug menu → Open Execute Anonymous Window (or Ctrl+E)

  3. 3

    Run Product Creation Script

    Copy the contents of createProductsAndPriceBook.apex and execute

  4. 4

    Verify in Setup

    Go to Setup → Products → Verify all 9 products are created and active

  5. 5

    Check Price Book

    Go to App Launcher → Price Books → Standard Price Book → Verify entries

  6. 6

    Add Products Related List to Opportunity Layout

    Setup → Object Manager → Opportunity → Page Layouts → Add "Products" related list

  7. 7

    Test with Sample Opportunity

    Create a test Opp → Click "Add Products" → Select products → Save → Verify

Validation Queries

Run these in Developer Console → Query Editor to verify setup:

-- Check Products Created
SELECT Id, Name, ProductCode, Family, IsActive 
FROM Product2 
WHERE IsActive = true
ORDER BY Family, Name
-- Check Price Book Entries
SELECT Product2.Name, Pricebook2.Name, UnitPrice, IsActive
FROM PricebookEntry
WHERE IsActive = true
ORDER BY Product2.Family, Product2.Name
-- Check Opportunity with Products (after testing)
SELECT Id, Name, Amount,
    (SELECT Product2.Name, Quantity, UnitPrice, TotalPrice 
     FROM OpportunityLineItems)
FROM Opportunity
WHERE Name LIKE '%Test%'
4

Opportunity Page Layout

  1. 1

    Navigate to Page Layouts

    Setup → Object Manager → Opportunity → Page Layouts

  2. 2

    Edit the Layout

    Click on your active layout (e.g., "Opportunity Layout")

  3. 3

    Add Products Related List

    Scroll to Related Lists → Drag "Products" to the layout → Position below key fields

  4. 4

    Configure Related List Columns

    Click wrench on Products list → Add: Product Name, Quantity, Sales Price, Total Price

  5. 5

    Save Layout

    Save the layout changes

5

Rollback Plan

If issues occur, deactivate products (doesn't delete existing data):

// Deactivate all Eudia products
List<Product2> products = [
    SELECT Id, IsActive 
    FROM Product2 
    WHERE ProductCode IN (
        'CONT-INH','CONT-MS','SIGMA','AGENTS',
        'MA-INH','MA-MS','COMP-INH','LIT','CONFIG'
    )
];

for (Product2 p : products) {
    p.IsActive = false;
}
update products;

System.debug('Deactivated ' + products.size() + ' products');
