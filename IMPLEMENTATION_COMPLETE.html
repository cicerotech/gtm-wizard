üîß How It Works

Assignment Algorithm

  1. Clay Enrichment: Retrieve company data (HQ, revenue, etc.)
  2. Region Detection: Determine West Coast / East Coast / International
  3. Get BL List: Find BLs responsible for that region
  4. Assess Workload: Query Salesforce for each BL's current coverage
  5. Calculate Score: (Active Opps √ó 3) + Closing This Month
  6. Select BL: Choose BL with LOWEST score
  7. Create Account: Save to Salesforce with selected owner
  8. Confirm: Show reasoning and assignment details

Geographic Mapping

Region States / Countries Assigned BLs
West Coast CA, OR, WA, NV, AZ, CO, UT + Central US Himanshu Agarwal, Julie Stefanich, Justin
East Coast NY, MA, FL, VA, etc. (all East Coast states) Olivia Jung
International Any country outside USA Johnson, Hannah

Workload Scoring

Formula: Score = (Active Stage 1+ Opps √ó 3) + Closing This Month

Why weighted 3:1? Active pipeline is the primary workload indicator. Deals closing this month are important but temporary, so weighted lower.

Selection: BL with LOWEST score gets the new account (fairest distribution)
