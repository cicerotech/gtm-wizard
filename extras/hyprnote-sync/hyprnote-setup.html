hyprnote setup

AI meeting notes for Eudia

Sales Team
Internal Use
1 Install Hyprnote

Download and install the app.

Download for Mac

Click "Hyprnote" (not Nightly). Drag to Applications. Open and sign in.

Done

Before calls: Hyprnote > New Note > Record

2 Open Terminal

Press Cmd + Space, type Terminal, press Enter.

3 Install Node.js

Paste this command and press Enter:

curl -o ~/Downloads/node.pkg https://nodejs.org/dist/v20.11.0/node-v20.11.0.pkg && open ~/Downloads/node.pkg

Click Continue through the installer. Enter your Mac password if asked.

4 Download Sync Tool

Paste this command and press Enter:

cd ~/Documents && curl -L -o hyprnote-sync.zip https://github.com/cicerotech/gtm-wizard/releases/download/v1.0.0/hyprnote-sync.zip && unzip -o hyprnote-sync.zip && rm hyprnote-sync.zip
5 Add Salesforce Credentials

Create the .env file with credentials (get password from Keigan):

cd ~/Documents/hyprnote-sync && cat > .env << 'EOF' SF_USERNAME=sync.service@eudia.com SF_PASSWORD=ASK_KEIGAN_FOR_PASSWORD SF_SECURITY_TOKEN= SF_INSTANCE_URL=https://eudia.my.salesforce.com EOF

Important: Replace ASK_KEIGAN_FOR_PASSWORD with the actual password from Keigan on Slack.

6 Run Setup

Paste this command and press Enter:

cd ~/Documents/hyprnote-sync && npm install --silent && node setup-quick.js

Confirm your name when prompted. Wait for "Setup Complete".

7 Test Sync

Run a test sync to verify everything works:

cd ~/Documents/hyprnote-sync && npm run sync

Should show "Connected!" and list your recent meetings.

Done

Run npm run sync after meetings to sync to Salesforce.

Daily Use
Before call Hyprnote > New Note > Record
