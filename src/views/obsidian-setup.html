A vault is just a folder on your computer where notes are stored. Each note is a simple text file.

Install Scribe Plugin
‚ñ∂

Scribe records your meetings and uses AI to transcribe and summarize them.

  1. In Obsidian, open Settings (gear icon, bottom-left)
  2. Go to Community plugins
  3. Click Turn on community plugins (if prompted)
  4. Click Browse
  5. Search for "Scribe"
  6. Click Install, then Enable

After installing Scribe, you must add our OpenAI API key for transcription to work.

  1. Go to Settings ‚Üí Scribe (in the left sidebar under Community plugins)
  2. Click the AI Providers tab
  3. Paste the OpenAI API key (provided by Keigan via Slack or during onboarding)
üîë Need the API key?

Ask @Keigan in Slack for the Eudia OpenAI API key. Do not share this key publicly.

  1. Set Transcript platform to: OpenAI
  2. Set LLM model to: gpt-4o
  3. Close Settings and restart Obsidian (Cmd+Q, then reopen)
‚úì That's it for core setup!

Scribe is now ready. You can record meetings and it will transcribe + summarize automatically.

3
Record Your First Meeting
‚ñ∂

Test that everything works by recording a quick test.

  1. Create a new note: Cmd + N
  2. Name it something like: Test Recording
  3. Open the Command Palette: Cmd + P
  4. Type "Scribe" and select "Scribe: Start recording"
  5. Speak for 10-15 seconds (e.g., "This is a test recording to make sure transcription works.")
  6. Open Command Palette again (Cmd + P) ‚Üí "Scribe: Stop recording"
  7. Wait a few seconds ‚Äî Scribe will transcribe and summarize your audio
üí° Pro tip: Organizing notes by account

Create a folder for each account (right-click in sidebar ‚Üí New folder). Put meeting notes inside the account folder. Example: Salesforce/2026-01-28 Discovery Call

4
Optional: Calendar Sync
‚ñ∂

Connect your Outlook calendar to see meetings directly in Obsidian.

Get Your Calendar URL

  1. Go to outlook.office365.com/calendar
  2. Click Settings (gear icon) ‚Üí View all Outlook settings
  3. Go to Calendar ‚Üí Shared calendars
  4. Under "Publish a calendar", select your calendar
  5. Set permissions to "Can view all details"
  6. Click Publish
  7. Copy the ICS link (not HTML)

Install Full Calendar Plugin

  1. Settings ‚Üí Community plugins ‚Üí Browse
  2. Search "Full Calendar" ‚Üí Install ‚Üí Enable
  3. Go to Settings ‚Üí Full Calendar
  4. Under Manage Calendars, click Add calendar
  5. Select Remote (.ics format)
  6. Paste your ICS URL and enable it
üéâ You're all set!

Your calendar will now appear in Obsidian. Create notes for meetings, record with Scribe, and organize by account.

Troubleshooting

Recording doesn't transcribe?

  • Make sure you restarted Obsidian after adding the API key
  • Check Settings ‚Üí Scribe ‚Üí AI Providers: API key should be filled
  • Check Settings ‚Üí Scribe ‚Üí AI Providers: Transcript platform = OpenAI

Microphone not working?

  • System Settings ‚Üí Privacy & Security ‚Üí Microphone
  • Make sure Obsidian is allowed
